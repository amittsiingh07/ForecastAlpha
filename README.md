# 🛰️ ForecastAlpha: Cross-Platform Sentiment Intelligence

**ForecastAlpha** is a high-density financial terminal designed to audit global crowd intelligence. By synchronizing real-time data from **Polymarket** and **Manifold**, the system identifies "Statistical Anomalies"—points where crowd sentiment deviates significantly from historical reality.

## 🚀 Key Features

- **Crowd Tension Index:** A proprietary volatility metric measuring the delta between crypto-native and play-money sentiment.
- **Automated Brier Audits:** Systematic calculation of crowd accuracy to identify overconfident market nodes.
- **Data Normalization:** A custom mapping layer that resolves raw API identifiers into clean, executive-ready outcomes.

## 🛠️ Tech Stack

- **Frontend:** Next.js (React), GSAP Animations, Geist Typography.
- **Backend:** FastAPI (Python), Uvicorn.
- **Database:** Supabase (PostgreSQL) with Real-time Persistence.
